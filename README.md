## Steps on how to create Github Page
1. Marke sure that you are in the correct repository that you want to publish, and that it has visibility set to public
2. On your Github repository, go to settings. Make sure that you are in the correct repository you want to publish
3. Under Code and Automation, click Pages. 
4. Under the Build and Deployment section, choose the branch where your index.html resides. In this case, we choose main. 
5. Save it, then after some waiting you will see the that your site is now live with Github Pages and you can click to view it.
